# Kaduvaprasad77 - Premium Design Studio

A stunning, fully animated website built with Next.js, React, Framer Motion, and Tailwind CSS. Inspired by Apple's design philosophy with smooth, sophisticated animations.

## 🌟 Features

- ✨ **Smooth Animations**: Buttery-smooth transitions powered by Framer Motion
- 📱 **Responsive Design**: Perfect on all devices
- ⚡ **Performance Optimized**: 60fps animations
- 🎨 **Modern UI**: Clean, Apple-inspired design
- 🚀 **SEO Optimized**: Google-indexed with sitemap and robots.txt

## 🌐 Live Website

**https://kaduvaprasad77.github.io**

## 🛠️ Tech Stack

- Next.js 14
- React 18.2
- Framer Motion 10.16
- Tailwind CSS

## 📦 Installation

```bash
npm install
npm run dev
```
